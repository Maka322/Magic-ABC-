# 🚀 ABC Mágico - La Lotería de las Letras 🚀 

¡Bienvenido al **ABC Mágico**! Una aplicación web interactiva diseñada para ayudar a los niños a aprender el abecedario y las vocales de una forma divertida y visual.

---

## Características
- Aprendizaje Visual: Cada letra muestra una imagen representativa (Ángel, Barco, Casa, etc.).
- Modos de Juego: Filtro rápido para ver solo las Vocales o el Abecedario completo.
- Contador de Vistas: Seguimiento de cuántas letras has descubierto.
- Diseño Colorido: Interfaz amigable y fácil de usar para los más pequeños.

## Tecnologías Usadas
- HTML5: Estructura del sitio.
- CSS3:Diseño responsivo, Flexbox y Grid.
- JavaScript:Lógica de filtrado, manejo del DOM y dinamismo.
- Git/GitHub: Control de versiones y trabajo colaborativo.

## Estructura del Proyecto
-  `index.html`: Estructura principal.
- `style.css`: Estilos, colores y animaciones.
- `app.js`: El "cerebro" puesto de mando y control
- `/img`: Carpeta que contiene las imagenes de cada letra.
